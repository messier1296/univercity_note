1. プログラムの説明

以下に，乱数を生成し，その値に対応する長さの棒を
アスタリスクで表示するプログラムを示す。
このプログラムでは，N 個の乱数を作り，それぞれを
0 以上 M 以下の整数に変換して配列 array に保存する。
その後，配列の各要素について，値の個数だけ "*" を
表示することでヒストグラムを作成する。


105.c

#include <stdio.h>
#include <stdlib.h>

#define M 30
#define N 1000

int main(void) {
  int array[N];
  srand(1);

  for (int i = 0; i < N; i++) {
    int j = rand();
    array[i] = 0 + (j * (M + 1.0) / (RAND_MAX + 1.0));
    printf("%d,", array[i]);
  }
  printf("\n \n");
  for (int i = 0; i < N; i++) {
    for (int j = 0; j < array[i]; j++) {
      printf("*");
    }
    printf("\n");
  }
}
