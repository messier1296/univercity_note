## 例：全国の小学生の身長の平均を知りたい

- ある小学校の身長から，全国の小学生の身長を推測すればよい
- **母集団**：知りたいものの全体の集まり
  - ここでは全国の小学生
- **母数**：知りたいものの特性値
  - 全国の小学生の平均身長，分散，標準偏差，最頻値等
- **標本集団**：実際に調べられるもの
  - ここではつくば市のある小学校の小学生

## 推測統計とは

- **標本集団** から **母集団** の **パラメータ** を **推定** すること

> We infer parameter from sample!

- 母数に具体的に数値を与える
  - 母数を **推定** する，estimate parameter.
- 与えられた具体的な数値
  - **推定量**，estimator.

算術平均 ＝ 全員の身長の和 / 全員の人数

- つくば市のある小学校の小学生の身長の算術平均は，全国の小学生の身長の平均値の推定量である

## 標本抽出の方法（sampling）

- 母集団から偏りのない標本を選択するための方法

### **単純無作為抽出法**

- すべての抽出の基本
- 母集団に含まれる全員に一連の通し番号をつけ，くじ引きのように誰もが等確率で当たるようにする方法
- 乱数表を用いてサンプルを選ぶ

### **等間隔ランダム法**

- 母集団に一連の通し番号をつけ，初めの1つだけ乱数で選び，後のサンプルは一定間隔（例えば20ごと）で選ぶ

### **多段階抽出法**

- 例：全国規模調査の場合
- いきなり全国民からサンプルを選ぶのではなく，まず市町村を選ぶ
- そして，選ばれた市町村の中から，実際の調査対象となる世帯を選ぶ方法
- 世論調査（政党支持など）でよく使われる
- 利点：大規模調査では，時間，労力，費用の節約になる
- 欠点：何段階にもわたり抽出を繰り返すので，母数の推定誤差が大きくなる

### **層別抽出法**

- 例：ある都市の医師の意識調査
- 被調査者を無作為に抽出しても，母集団の構成（専門科別，開業医別）がサンプルに反映されないことがある
- あらかじめ医師を専門科別，開業医別のような等質のグループに分け，それぞれのグループから標本を抽出する方法
- 世論調査（消費行動など）でもよく使われる

## ランダムサンプリングの重要性

- ランダムサンプリングをしないと，正しい統計的推測ができない
- もし母集団が偏っていた場合どうなるのか
  - 一例：1936年のアメリカ大統領選
  - Literary Digestの調査（200万人）
    - 対立候補の勝利を予想（57%対43%）
  - ギャラップの調査（30万人）
    - ルーズベルトの勝利を予想
  - 結果は，**ギャラップ** の調査が正しかった
- Literary Digestは電話や自動車の保有者リストを用いたため，標本サイズは莫大だったが，データが富裕層に偏ってしまった
- このことは，雑誌の読者アンケートやWeb調査にもいえるので注意が必要

## 仮説検定（Hypothesis Testing）

- 母集団のパラメーターがある値であるかどうか（**仮説**）の **妥当性** を調べる方法

> whether the results from sample valid or not

- ここでは，つくば市のある小学校の小学生の身長の算術平均が，全国の小学生の身長の平均に近いかどうかの妥当性を与えること
- 例：新製品が従来製品よりも優れているかどうか

## 推定と検定（estimate and test）

- 推定と検定は，統計的推測の二本柱
- 標本に基づいて，母集団のパラメータに関する推論をおこなう典型的な方法
- 統計的推測の理論の根底には，確率と確率変数の概念がある
- 推定や検定においても，確率と確率変数の概念を理解する必要がある

## 統計データ（statistics data）

- 統計データとは，何らかの変動する要因（変数または変量，variable）に関して，現実の事象や集団を測定または調査して得られたもの
- **量的** データと **質的** データがある

### **量的** データ（quantitative data）

- **数値の大小** に意味があるデータ
- **離散** 変数（discrete variable）
  - 事故件数，家族の人数など（少数の値をとらない）
- **連続** 変数（continuous variable）
  - 重さ，長さ，為替レートなど（少数の値をとる）
- ただし離散変数を連続変数として扱う場合も多い

### 量的データの尺度

- **量的** データにおいて，0 = nothing の場合とそうでない場合がある
- **比率** 尺度（ratio scale）
  - 0 = nothing
  - 長さ，金額など0が絶対的な意味を持つ（無を表す）
  - 100kgは50kgの2倍といえる
  - つまり掛け算，割り算もできる
- **間隔** 尺度（interval scale）
  - 0 ≠ nothing
  - 温度，学力，IQなど
  - 例えば英語0点だからといって，英語の能力がnothingであるわけではない
  - 足し算，引き算はできる

### **質的** データ（quality data）

- **順位尺度** データ
- **順位** 尺度（ordinal scale）
  - 大小関係が存在するもの
  - 成績（優，良，可，不可），地震の震度など
- **名義** 尺度（nominal scale）
  - 大小関係が存在しないもの
  - 性別，職業，TVのchannelなど

## 尺度とその種類

| Numerical value? | データ型 | 尺度 | 例 |
| --- | --- | --- | --- |
| Yes | quantitative data | ratio scale | Weight, length |
| Yes | quantitative data | interval scale | Temperature, IQ |
| No | quality data | ordinal scale | Record, seismic intensity |
| No | quality data | nominal scale | TV channel |

## 推測統計（inferential statistics）と確率モデル（probabilistic model）

- 全数調査のように母集団の中身がすべてわかれば，推測統計はいらない
- しかし現実には，経済，心理，社会データのように標本調査をせざるを得ない場合が多い
- 例：正確な重さを測定しにくい物体を測定する場合
- 一回では正確な真の重さを測定できないため，何回か重さを測定して真の値を測定する必要がある
- この場合のモデルは以下のように表現可能

測定値 ＝ 真の重さ ＋ 測定誤差  
Observation = True mean + Error

## 一般的な確率モデル

- 確率変数 ＝ 系統的部分 ＋ 誤差項

> Random variable = Systematic part + Error

- 確率変数：とりうる値の全体がわかっており，固有の確率法則にしたがって実現するような変数（実際に観測される変数）
- 系統的部分：物体の重さのような未知の定数
- 誤差項：系統的部分では説明不可能なすべての要因からなる，測定や実験のたびに変動する確率変数
- 系統的部分，誤差項は分析者が合理的と考えるパラメータを導入してモデル化される

## なぜ確率モデルにするのか

- 測定値を **確率** 変数 ＝ **系統的** 部分 ＋ **誤差項** 項 に分解して，確率モデルにする
- 母集団を単純化して，推定や検定などを容易にする
- さらにどのようなデータが出るかという推測に便利

> Population is simplified, and estimation and testing become easy. It is more convenient for the predictive.

- 確率モデルの例
  - 回帰モデル（regression model）：給与所得から小遣いを予測
  - 時系列モデル：株価や為替レートの変動を予測

- `t(7) = -5.586, p = .001` とは，自由度7の t 分布で t 値が `-5.586` のとき，**両側の面積** が `.001` という意味
- 簡単に言うと，**2つの母平均が等しい** という確率が **0.01** しかないということ

## 仮説の検定

- 2種類の過誤がある

### **第一種** の過誤

- 仮説 H0（A = B や A = B = C など）について，母平均に **差がない** にもかかわらず **差がある** という判定をすること
- 言い換えると，仮説 H0 が正しいにもかかわらず，仮説 H1 を採択する誤り
- 例
  - H0：ウェイトレスの時給 ＝ コンパニオンの時給
  - これが正しいにもかかわらず，
  - H1：ウェイトレスの時給 ≠ コンパニオンの時給
  - を採択する誤り

### **第二種** の過誤

- 2つの母平均に **差がある** にもかかわらず **差がない** という判定をすること
- 言い換えると，仮説 H1 が正しいにもかかわらず，仮説 H0 を採択する誤り
- 例
  - H1：ウェイトレスの時給 ≠ コンパニオンの時給
  - これが正しいにもかかわらず，
  - H0：ウェイトレスの時給 ＝ コンパニオンの時給
  - を採択する誤り
- この2つの誤りを同時に小さくすることは不可能

## 有意水準

- 通常は第一種の過誤を小さくすることによって，仮説 H0（母平均が等しい）かどうかを検定する
- 有意水準は主に `p < .05`，`p < .01`，`p < .001` の3つ
- 基本的に，第一種の過誤が起こる確率が `p < .05`（有意水準5%）であれば，有意に差があったとみなす
- `p < .01` や `p < .001` があるのは，自分が立てた仮説が正しいことをより強く主張するため
- `p < .05` よりも小さい値が出た場合は，`p < .01` や `p < .001` を使って，1%水準あるいは0.1%水準で有意だったと書く
