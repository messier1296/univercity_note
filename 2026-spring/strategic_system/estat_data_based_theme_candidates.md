# e-Stat公開データから考えるテーマ候補

e-Statで公開されている統計データをジャンル別に探索し、各ジャンルからテーマ候補を5個ずつ列挙する。
同じ統計で別の年のデータは、1つのデータとして扱う。

## 1. 人口・世帯

探索したデータ: 国勢調査、人口推計、住民基本台帳人口移動報告、人口動態調査、国民生活基礎調査

1. 市区町村別に見る単身世帯割合と高齢化率の関係
2. 都道府県別の若年人口割合と転入超過率の関係
3. 人口減少が大きい市区町村に共通する世帯構成の特徴
4. 出生率が低い都道府県における未婚率と平均世帯人員の関係
5. 外国人人口割合が高い市区町村の年齢構成の特徴

## 2. 人口移動・地域定着

探索したデータ: 住民基本台帳人口移動報告、国勢調査、人口推計、在留外国人統計、出入国管理統計

1. 20代の転入超過が大きい都道府県に共通する産業構成
2. 大都市圏への若年層流入と地方県の高齢化率の関係
3. 大学進学年齢層の人口移動が多い地域の特徴
4. 外国人住民の増加率が高い市区町村の人口減少緩和効果
5. 転出超過が続く自治体における世帯数変化の特徴

## 3. 労働・雇用

探索したデータ: 労働力調査、就業構造基本調査、雇用動向調査、毎月勤労統計調査、大学等卒業予定者の就職内定状況等調査

1. 都道府県別の完全失業率と若年人口割合の関係
2. 女性の就業率が高い地域における非正規雇用割合の特徴
3. 産業別の入職率と離職率から見る人手不足業種
4. 新卒内定率の変化と景気指標の関係
5. 高齢者就業率が高い都道府県の産業構成の特徴

## 4. 賃金・労働条件

1. 都道府県別の平均賃金と女性就業率の関係
2. 産業別の所定内給与額と離職率の関係
3. 企業規模別に見る賃金格差と賞与額の違い
4. 年齢階級別の賃金カーブが急な産業の特徴
5. 労働時間が長い産業ほど賃金水準は高いのか

## 5. 家計消費

探索したデータ: 家計調査、全国家計構造調査、家計消費状況調査、全国単身世帯収支実態調査、消費者物価指数

1. 都市別の外食支出額と共働き世帯割合の関係
2. 単身世帯と二人以上世帯で異なる食費構成の特徴
3. 物価上昇期に減りやすい娯楽支出の品目

5. 若年単身世帯の通信費支出が消費全体に占める割合
